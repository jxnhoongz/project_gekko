## 3. Tech decisions (decided, locked)

| Layer | Choice | Concrete details |
|---|---|---|
| **Repo structure** | **Monorepo** via bun workspaces | Package root at repo root with `workspaces: ["apps/*"]`. Backend is NOT a workspace (Go uses its own module system); it sits at `backend/` at the repo root. |
| **Package manager** | **[bun](https://bun.sh/)** ≥ 1.1 | One lockfile (`bun.lock`) at repo root covers all Vue workspaces. No `npm`, `yarn`, or `pnpm`. |
| **Backend language** | **Go** ≥ 1.23 (rewrite from Node/Hono) | Single binary, single service, single Go module. Not microservices. |
| **Backend HTTP framework** | **[chi](https://github.com/go-chi/chi) v5** | Idiomatic net/http-compatible router. Minimal, middleware-friendly. Install: `go get github.com/go-chi/chi/v5`. |
| **Go DB driver** | **[pgx](https://github.com/jackc/pgx) v5** | Modern, native Postgres driver. Install: `go get github.com/jackc/pgx/v5`. Use `pgxpool.Pool` for connection pooling. |
| **Go SQL toolkit** | **[sqlc](https://sqlc.dev/) v1.27+** | Generates type-safe Go code from raw SQL queries. Query files live in `backend/internal/queries/*.sql`, generated code lands in `backend/internal/db/`. |
| **Go migrations** | **[goose](https://github.com/pressly/goose) v3** | File-based migrations in `backend/migrations/`. Format: each file has `-- +goose Up` / `-- +goose Down` sections. |
| **Go dev hot reload** | **[air](https://github.com/air-verse/air) v1.52+** | Watches Go files, rebuilds + restarts. Config in `backend/.air.toml`. |
| **Go logging** | **[zerolog](https://github.com/rs/zerolog) v1.33+** | Structured JSON logs in prod, pretty console in dev. Install: `go get github.com/rs/zerolog`. |
| **Go JWT** | **[golang-jwt/jwt](https://github.com/golang-jwt/jwt) v5** | `go get github.com/golang-jwt/jwt/v5`. |
| **Go validator** | **[go-playground/validator](https://github.com/go-playground/validator) v10** | `go get github.com/go-playground/validator/v10`. Tag-based struct validation. |
| **Database** | **Postgres 15** | Local: Docker container. Prod: managed Postgres on chosen host. |
| **Admin frontend** | **Vue 3.5 + Vite 7 + TypeScript 5 + Tailwind 4** | `bunx create-vite@latest apps/admin -- --template vue-ts`. |
| **Admin component library** | **[shadcn-vue](https://www.shadcn-vue.com/)** (Reka UI) | Install via `bunx shadcn-vue@latest init`. Theme via CSS variables — see `gekko-design` skill. |
| **Admin data fetching** | **[@tanstack/vue-query](https://tanstack.com/query/latest/docs/framework/vue/overview) v5** | `bun add @tanstack/vue-query`. Handles caching, loading states, refetch on focus. |
| **Admin table** | **[@tanstack/vue-table](https://tanstack.com/table/latest) v8** | For the Data visualizer + waitlist list. `bun add @tanstack/vue-table`. |
| **Admin forms** | **vee-validate + zod** | `bun add vee-validate @vee-validate/zod zod`. |
| **Admin toasts** | **vue-sonner** | `bun add vue-sonner`. |
| **Admin icons** | **lucide-vue-next** | `bun add lucide-vue-next`. |
| **Admin HTTP client** | **axios** | `bun add axios`. JWT bearer attached via interceptor (Pinia auth store). |
| **Storefront** | **Vue 3 + Vite + Vue I18n** | Mostly unchanged from legacy. Currently on Vercel. |
| **Photo storage** | v1: backend filesystem · v2: Cloudflare R2 | See design spec §10. |
| **Backend hosting** | **TBD** (Railway / Fly.io / VPS) | Answered in §11 open decision #3 on Linux. |
| **Reverse proxy (prod)** | **[Caddy](https://caddyserver.com/) v2** | One `Caddyfile` handles TLS (auto Let's Encrypt), routes `api.zeneticgekkos.com` → Go backend, static files for admin/storefront. |
| **Container runtime** | **Docker + Docker Compose v2** | Local dev Postgres. Optional prod deployment. |
| **Mobile apps** | **None.** Responsive web only. | Explicitly no App Store / Play Store. |

### Stack decisions REJECTED (and why)

- **Node/Hono/Drizzle backend** → user wants a single Go toolchain across personal projects → Go. ~1500 lines of existing Node code becomes reference material only.
- **Microservices** → massive overkill for 6 geckos + a waitlist. One Go monolith.
- **gRPC** → no service-to-service calls to justify it. Plain REST JSON.
- **GORM / ent** (Go ORMs) → sqlc produces simpler, inspectable Go from hand-written SQL. We own the queries.
- **Flutter mobile app** → explicitly rejected. Web only.
- **Telegram as image CDN** → rejected. URLs unstable, ToS risk, no transformations.
- **pnpm / yarn / npm** → chose bun for speed and simpler single-tool story.

### 9.2 Local Postgres first

Before any app needs the DB, get Postgres running:

```bash

services:
  postgres:
    image: postgres:15-alpine
    container_name: gekko_db
    restart: unless-stopped
    environment:
      POSTGRES_DB: gekko
      POSTGRES_USER: gekko
      POSTGRES_PASSWORD: gekko
      POSTGRES_INITDB_ARGS: "-E UTF8 --locale=en_US.UTF-8"
    ports:
      - "5433:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U gekko"]
      interval: 10s
      timeout: 5s
      retries: 5

volumes:
  postgres_data:
EOF

docker compose -f infra/docker-compose.dev.yml up -d

# Wait for healthy, then test connection
until docker exec gekko_db pg_isready -U gekko >/dev/null 2>&1; do sleep 1; done
echo "Postgres ready on localhost:5433"

# Optional: sanity check from host if psql is installed
PGPASSWORD=gekko psql -h localhost -p 5433 -U gekko -d gekko -c '\dt'
```

	_ = godotenv.Load(".env.local")

	if os.Getenv("LOG_LEVEL") == "debug" {
		log.Logger = log.Output(zerolog.ConsoleWriter{Out: os.Stderr, TimeFormat: time.Kitchen})
	}
	zerolog.TimeFieldFormat = zerolog.TimeFormatUnix

	ctx := context.Background()
	pool, err := pgxpool.New(ctx, os.Getenv("DB_URL"))
	if err != nil {
		log.Fatal().Err(err).Msg("db connection failed")
	}
	defer pool.Close()

	r := chi.NewRouter()
	r.Use(chimw.RequestID, chimw.RealIP, chimw.Logger, chimw.Recoverer)
	r.Use(cors.Handler(cors.Options{
		AllowedOrigins:   splitCSV(os.Getenv("CORS_ORIGINS")),
		AllowedMethods:   []string{"GET", "POST", "PUT", "PATCH", "DELETE", "OPTIONS"},
		AllowedHeaders:   []string{"Accept", "Authorization", "Content-Type"},
		ExposedHeaders:   []string{"Link"},
		AllowCredentials: true,
		MaxAge:           300,
	}))

	r.Get("/health", func(w http.ResponseWriter, req *http.Request) {
		if err := pool.Ping(req.Context()); err != nil {
			http.Error(w, `{"status":"db down"}`, http.StatusServiceUnavailable)
			return
		}
		w.Header().Set("Content-Type", "application/json")
		w.Write([]byte(`{"status":"healthy"}`))
	})

	// Serve uploaded photos as static files (v1)
	uploadDir := os.Getenv("UPLOAD_DIR")
	if uploadDir == "" {
		uploadDir = "./uploads"
	}
	fs := http.StripPrefix("/uploads/", http.FileServer(http.Dir(uploadDir)))
	r.Handle("/uploads/*", fs)

	// TODO: mount api routes here once handlers are written
	// r.Route("/api", routes.Mount(pool))

	port := os.Getenv("PORT")
	if port == "" {
		port = "8420"
	}
	log.Info().Msgf("listening on :%s", port)
	if err := http.ListenAndServe(":"+port, r); err != nil {
		log.Fatal().Err(err).Send()
	}
}

func splitCSV(s string) []string {
	out := []string{}
	cur := ""
	for _, c := range s {
		if c == ',' {
			if cur != "" {
				out = append(out, cur)
			}
			cur = ""
			continue
		}
		cur += string(c)
	}
	if cur != "" {
		out = append(out, cur)
	}
	return out
}
